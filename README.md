# FontLoaderSubRe

Heavily inspired by [FontLoaderSub](https://github.com/yzwduck/FontLoaderSub).

Drag-and-drop ASS/SSA subtitles and it will load corresponding font files from a font base.

## Usage

1. Open FontLoaderSubRe, set font directory path, index the fonts from menu. (Only needed first time)
2. Drag-and-drop subtitles `*.ass` `*.ssa` or folders onto the `FontLoaderSubRe` window.

## Changes

* Redesigned with Python and PySide6.

### New Features

* Extract fonts and subtitles from MKV (needs to install [MKVToolNix](https://mkvtoolnix.download/downloads.html)).
* Load font files.
